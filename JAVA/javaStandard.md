1. 每一个源代码前需要有解释整个源代码的注释。每一个类或方法前需要有注释，没做完的需要有*TODO*，*TODO*后有解释。
```java
// 这是一段注释
class javaHelloWorld {
    public static void main(String args[]){
        System.out.println("hello world\n");
    }
}

// TODO main方法没有写完，缺少了输出
class javaHelloWorld {
    public static void main(String args[]){
        
    }
}
```
